# Ripple React App

This project rebuilds the original Mini Social Networking App as **Ripple**, using a React/Vite frontend with locally persisted social features translated from `Ripple_nexo_full_backend_merge`.

# Live Preview

https://ripple-react.vercel.app/

## Included feature set

- Ripple brand, palette, and glassmorphism-inspired shell
- Login and registration flows
- Feed composer with draft autosave
- Post create, edit, delete, like, save, and comment CRUD
- Saved posts view
- Friend requests and accepted-friend management
- Friends-only post visibility
- Notifications center
- Messenger-style chat with replies, media links, and quick replies
- Theme persistence with auto/light/dark modes
- Feed filters and density modes
- Per-user local persistence via browser storage

## Running the app

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
